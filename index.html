Description :

Pour les commandes, faites .help


Si vous avez des question par rapport au bot, veuillez me contacter sur discord Fizycol_Namania#7702

Important ! ne supprimer pas le rôle Muted que le bot créer automatiquement !


Commands :

.ban -> pour ban des membre si vous avez la permissiona 'BAN_MEMBERS'

.cl -> pour clear les messages si vous avez la permissiona 'ADMINISTRATOR'

.help -> pour voir les commandes disponible

.invite -> pour avoir le lien d'invitation du bot

.links -> pour mes réseaux

.mc -> pour mes mods, rescources | shaders packs

.mute -> pour mute des membre si vous avez la permissiona 'BAN_MEMBERS'

.ping -> pour voir son ping

.server -> pour avoir des informations sur le server

.setup -> pour choisir dans quelle channel le bot peux envoyer ses messages

.sw -> pour activer ou désactiver des commandes (toute les commandes sont active de bases)

.tempmumte -> pour mute des membre pendant une durée prédéfini si vous avez la permissiona 'BAN_MEMBERS'

.ticket -> pour faire un retour | une demande | une plainte

.unban -> pour unban des membre si vous avez la permissiona 'BAN_MEMBERS'

.unmute -> pour unmute des membre si vous avez la permissiona 'BAN_MEMBERS'


Pour la commande ticket :

vous pouvez créer un ticket -> .ticket TITRE CONTENU

vous pouver édit un ticket -> .ticket edit TOCKET ( title | contenu ) ( Si title alors 1 seul argument ! )

vous pouvez supprimer un ticket -> .ticket rm TOCKEN

vous pouvez voir la list des ticket du server ( Si vous avez les permission 'ADMINISTRATOR' ) .ticket list
